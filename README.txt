WMA-RCA PEIC (Public Calculator of Environmental Impact).

All files and codes are available for download, use, modification, and distribution.

File listing:
"Methodology.ipynb" - It is a notebook developed in Google Colaboratory, in which the methodology developed (without the GUI) is presented.
"gui.pyw" - This file contains the proposed methodology within a GUI.
"icon.ico" - This is the icon created for the computational tool.
